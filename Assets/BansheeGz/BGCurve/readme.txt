Quick tips:
1) To create a curve chose GameObjecty->Create Other->BG Curve or AddComponent and search "BG Curve"
2) You can learn a lot by looking at fields tooltips and contextual tips in the editor
3) You can save & load curve settings on/from disk
4) You can select several points and perform group operations
5) BGCurve class does not contain any Math operations- to use Math- use BGCurveBaseMath or your own class (using BGCurveBaseMath as an example). BGCurveBaseMath is very basic , non optimized class.
